# Front-End
About
====
Celebrity Dead or Alive is a quiz.  Can you guess the status of all
10 celebrities?  Try it here.


Setup
===

tech installed using yarn add
*   axios
*   react
*   react-dom
*   react-router-dom
*   formik
*   react-redux
*   redux
*   yup
*   styled-components

Programs used to make the app

*   Visual Studio Code
